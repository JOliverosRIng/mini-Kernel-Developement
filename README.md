# Mini-Kernel: Simulación de Sistema Operativo

**Proyecto Final** — Sistemas Operativos 2026-1  
Universidad Distrital Francisco José de Caldas

## Descripción del Proyecto

Implementación modular de un **Mini-Kernel** educativo en C++20 que simula los componentes principales de un sistema operativo:

1. Fase 1: Gestión de Procesos
2. Fase 2: Gestión de Memoria
3. Fase 3: Sistema de Archivos y E/S
4. Demostración Completa: Kernel Integrado

## Requisitos

- Compilador con soporte C++20 (g++ 10+, clang 10+, MSVC 2019+)
- Make

## Compilación y Ejecución

```bash
make          # Compilar programa principal y tests
make run      # Ejecutar el mini_kernel con menú interactivo
make test     # Ejecutar  tests unitarios
make clean    # Limpiar ejecutables y archivos temporales
```
